# SpoutTokens

### Prerequisite

  - NodeJS >= 8.5.0

### Installation

  1. Install [truffle](https://github.com/trufflesuite/truffle)

  `$ npm install -g truffle`

  2. Compile & migrate

  ```sh
  // clone this repo first
  $ cd /path/to/project
  $ truffle compile
  $ truffle migrate --network [development|rinkeby]
  ```
